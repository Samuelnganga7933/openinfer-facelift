

OpenInfer Facelift is a web-based interface built on top of OpenInfer. It provides a clean and intuitive interface for running inferences, visualizing results, and managing workflows efficiently. The project is designed for flexibility, clarity, and ease of use.

## Features

- Fast and responsive interface
- Easy configuration for multiple inference models
- Minimalist design with Tailwind CSS
- Modular structure for future enhancements

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Samuelnganga7933/openinfer-facelift.git
cd openinfer-facelift

2. Install dependencies:



npm install

3. Start the development server:



npm run dev

4. Open your browser at http://localhost:3000 to access the interface.



Usage

Connect the project to your inference models

Run predictions and view outputs through the interface

Extend functionality by adding custom modules or integrating additional models


Project Structure

tailwind.css – Interface styling

vite.config.js – Development server configuration

.eleventy.js – Static site generator setup

package.json – Dependencies and scripts
